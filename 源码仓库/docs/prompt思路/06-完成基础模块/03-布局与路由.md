## AI 的回答

以下是按照你的要求，对前端布局与路由系统的完整实现。

### 布局组件

**TopHeader** (`components/TopHeader.tsx`)

- `variant="public"`：显示 Logo + 右侧"工作台"按钮（已登录用户点击进入 `/datasets`，未登录跳转 `/login`）
- `variant="workspace"`：显示 Logo + 用户头像及姓名下拉菜单 + "返回首页"按钮（切换到 `/`）
- 公共 header 出现在 `/`, `/login`, `/register`；工作台 header 出现在所有 `/datasets`, `/schemas`, `/tasks` 等子页

**AppLayout** (`pages/AppLayout.tsx`)

```tsx
const AppLayout = () => (
  <div className="app-layout">
    <TopHeader variant="public" />
    <main className="app-content">
      <Outlet />
    </main>
  </div>
);
```

公共页布局，包裹 `/`, `/login`, `/register` 三个页面。

**WorkspaceLayout** (`pages/WorkspaceLayout.tsx`)

```tsx
const WorkspaceLayout = () => {
  const user = useAuthStore(s => s.user);
  
  return (
    <Layout className="workspace-layout">
      <WorkspaceSidebar role={user!.role} />
      <Layout>
        <TopHeader variant="workspace" />
        <Layout.Content>
          <Outlet />
        </Layout.Content>
      </Layout>
    </Layout>
  );
};
```

工作台布局：左侧可折叠侧边栏 + 顶部工作台 header + 内容区。

**WorkspaceSidebar** 按角色渲染菜单：

| 角色 | 菜单项 |
|------|--------|
| Owner | 数据集 \| Schema设计器 \| 任务管理 \| AI配置 \| AI预审 \| 审核中心 \| 看板 \| 导出历史 |
| Reviewer | AI预审 \| 审核中心 |
| Labeler | 标注工作台 \| 我的贡献 |

侧边栏根据 `user.role` 动态生成菜单，当前选中项通过 `useLocation()` 匹配高亮。

### 路由守卫

**ProtectedRoute** (`components/ProtectedRoute.tsx`)

```typescript
const ProtectedRoute = () => {
  const { isLoggedIn, authReady } = useAuthStore();

  if (!authReady) return <Spin size="large" tip="加载中..." />;
  if (!isLoggedIn) return <Navigate to="/login" replace />;

  return <Outlet />;
};
```

- `authReady === false`（正在验证 token）：显示全屏 Spin
- `isLoggedIn === false`：重定向到 `/login`
- 通过后渲染子路由

**RoleRoute** (`components/RoleRoute.tsx`)

```typescript
const RoleRoute = ({ roles }: { roles: string[] }) => {
  const user = useAuthStore(s => s.user);

  if (!user || !roles.includes(user.role)) {
    return <Result status="403" title="403" subTitle="无权访问此页面" />;
  }

  return <Outlet />;
};
```

角色不匹配时展示 403 页面，而非重定向，这样用户可以明确知道权限不足。

### 路由树 (`App.tsx`)

```tsx
<BrowserRouter>
  <Routes>
    {/* 公共页: AppLayout */}
    <Route element={<AppLayout />}>
      <Route path="/" element={<Home />} />
      <Route path="/login" element={<Login />} />
      <Route path="/register" element={<Register />} />
    </Route>

    {/* 工作台: ProtectedRoute → WorkspaceLayout */}
    <Route element={<ProtectedRoute />}>
      <Route element={<WorkspaceLayout />}>

        {/* Owner 专属 */}
        <Route element={<RoleRoute roles={["owner"]} />}>
          <Route path="/datasets" element={<Datasets />} />
          <Route path="/schemas" element={<SchemaDesigner />} />
          <Route path="/tasks" element={<TaskManage />} />
          <Route path="/ai-configs" element={<AiConfigs />} />
          <Route path="/dashboard" element={<Dashboard />} />
          <Route path="/export-history" element={<ExportHistory />} />
        </Route>

        {/* Owner + Reviewer 共享 */}
        <Route element={<RoleRoute roles={["owner", "reviewer"]} />}>
          <Route path="/review" element={<Review />} />
          <Route path="/ai-review" element={<AiReview />} />
        </Route>

        {/* Labeler 专属 */}
        <Route element={<RoleRoute roles={["labeler"]} />}>
          <Route path="/labeling" element={<Labeling />} />
          <Route path="/earnings" element={<Earnings />} />
        </Route>

      </Route>
    </Route>

    {/* 404 */}
    <Route path="*" element={<NotFound />} />
  </Routes>
</BrowserRouter>
```

嵌套守卫链：`ProtectedRoute`（登录检查）→ `WorkspaceLayout`（侧边栏+header）→ `RoleRoute`（角色检查）→ 页面组件。404 放在最外层兜底。
