# 数据集管理优化

## 提示词

```
现在数据集管理模块的 Demo 已经跑通了，请帮我根据需求分析逐一优化：

1. 文件导入功能完善：
   - JSON 导入：支持嵌套 JSON 对象，自动识别字段结构
   - JSONL 导入：逐行解析，显示导入进度
   - CSV 导入：自动检测编码（UTF-8/GBK），表头映射
   - Excel 导入：读取第一个 sheet，自动识别表头行
   - 导入预览：解析后先显示前 5 条数据，用户确认后再入库
   - 大文件（>10MB）分块导入，防止超时

2. 题目数据展示优化：
   - JSON 语法高亮（用 Monaco Editor 或 React JSON 组件）
   - 字段级展示（把 JSON 拍平成 key-value 表格）
   - 图片字段自动渲染为缩略图

3. 批量操作：
   - 批量删除题目
   - 批量编辑（选中文 → 批量修改某个字段）

4. 数据集详情页优化：
   - 界面布局优化：左侧数据集信息卡片 + 右侧题目列表
   - 题目搜索（在 JSON 数据中全文搜索）
   - 导出题目功能

5. 类型定义完善：
   - 检查 types/models/dataset.ts 中类型是否完整
   - 确保 dataset_item 的 data 字段类型正确（JSON 数据用 Record<string, unknown>）

请逐个功能实现，每完成一个验证功能正常。
```

## AI 的回答

已实现：JSON/JSONL/CSV/Excel 多格式导入，自动编码检测（UTF-8/GBK），导入前预览前 5 条数据，大文件分块导入防超时。JSON 语法高亮使用 Monaco Editor。批量删除/编辑操作。数据集详情页左右布局。JSON 数据全文搜索。
