# 标注工作台完善

## 提示词

```
请完善标注员工作台的以下功能：

1. Schema 驱动表单渲染完善：
   - 所有 10 种字段类型的 Renderer 都能在 Formily 中正确渲染
   - 字段级校验：必填/长度/正则/自定义
   - 条件显示（reactions）：某字段值变化时其他字段显隐
   - Tab 切换：有多个 Tab 时只渲染当前激活 Tab
   - 分组容器渲染为 Card/Fieldset 包裹形式

2. LLM 辅助功能完善：
   - 工作台中的 LLMInteraction 字段 → 点击"AI 建议"按钮
   - 后端调用 /api/llm-trigger：
     传入 task 信息 + item 原始数据 + 字段 key
     返回 LLM 生成的建议文本
   - 建议文本可一键填入对应字段

3. 草稿保存完善：
   - 自动保存策略：字段变更 3 秒后自动保存（debounce）
   - 保存时显示 loading 状态
   - 页面关闭前弹窗确认（beforeunload 事件）
   - 刷新页面后恢复上次草稿

4. 打回修改流程：
   - 被打回的题目在队列中显示特殊标识（红色标记）
   - 点击进入时显示打回原因 + 审核意见
   - 修改后重新提交

5. 我的数据页 (Earnings 页面)：
   - 统计卡片：已提交数、通过数、打回数
   - 提交记录列表：任务名、题目、提交时间、状态
   - 按状态筛选

6. 上一题/下一题/跳题：
   - 上一题：切换到上一题，保留草稿
   - 下一题：切换到下一题，自动保存当前草稿
   - 快捷跳题：点击 ItemQueue 中的任意题目

请按顺序完善，每完成一项在标注工作台实际操作验证。
