# 云平台部署

## 提示词

```
请为 LabelHub 编写云平台部署指南。

目标：部署到一台云服务器上，外网可访问。

1. 服务器环境准备：
   - 系统：Ubuntu 22.04 / CentOS 8
   - 安装 Docker + Docker Compose
   - 配置防火墙开放端口：80、443、5173（前端）、8000（后端 API）
   - 如果有域名，配置 DNS 解析

2. 部署步骤：
   a. 克隆代码仓库到服务器：
      git clone <repo-url> /opt/labelhub
   b. cd /opt/labelhub
   c. cp .env.example .env
   d. 编辑 .env：
      - DB_PASSWORD=<设定的密码>
      - JWT_SECRET=<随机生成的 32 位密钥>
      - LLM_API_KEY=<你的 API Key>
      - LLM_BASE_URL=<你的 API 地址>
   e. docker-compose up -d
   f. 等待服务启动 → 访问 http://<服务器IP>:5173

3. Nginx 反向代理（推荐）：
   - 创建 nginx.conf：
     server {
       listen 80;
       server_name labelhub.example.com;
       
       location / {
         proxy_pass http://127.0.0.1:5173;
       }
       
       location /api/ {
         proxy_pass http://127.0.0.1:8000;
       }
     }
   - 也可以用 docker-compose 加 nginx 服务

4. 初始化种子数据：
   docker-compose exec backend python -m app.scripts.seed

5. 健康检查：
   curl http://localhost:8000/api/health

6. 日志查看：
   docker-compose logs -f backend
   docker-compose logs -f frontend

7. 备份建议：
   - MySQL 数据库定期备份
   - 备份 .env 文件

请生成一份完整的部署文档（Markdown），包含每一步的命令和预期输出。
```
