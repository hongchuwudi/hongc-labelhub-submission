# AI 配置与审核监控模块 Demo

## 提示词

```
请帮我实现 AI Agent 配置和 AI 审核监控模块的 MVP 版本。

后端部分：
1. 创建 AiAgent 模型 (models/ai/agent.py)：
   - id, name, email(唯一), password_hash
   - system_prompt(TEXT), scoring_dimensions(JSON)
   - llm_model, created_by
   - 注意：AI Agent 同时也是一个 User（role=ai_agent）

2. AI Agent API (api/v1/ai/router.py)：
   - GET /api/ai-agents — 列表
   - POST /api/ai-agents — 创建 Agent（自动创建对应用户）
   - PUT /api/ai-agents/:id — 更新配置
   - DELETE /api/ai-agents/:id — 删除
   - GET /api/ai-agents/status — 获取 Agent 池运行状态

3. AI 审核记录模型 (models/tasks/ai_review.py)：
   - id, task_id, item_id, result_id(唯一), agent_id
   - verdict(pass/reject/human_review/error)
   - summary(TEXT), dimensions(JSON), overall_score
   - model, prompt_template, prompt_vars(JSON)
   - status(pending/processing/done/failed)
   - error_message, duration_ms
   - created_at, finished_at

4. AI 审核 API (api/v1/ai/reviews.py)：
   - GET /api/ai-reviews — 列表（支持 agent/verdict/status 筛选）
   - GET /api/ai-reviews/:id — 详情
   - POST /api/ai-reviews/:id/rerun — 失败重跑
   - GET /api/ai-reviews/agents-stats — 全部 Agent 统计
   - GET /api/ai-reviews/agent/:id/stats — 单 Agent 统计

前端部分：
1. AI 配置页 (pages/AiConfigs/index.tsx)：
   - Agent 列表表格：名称、模型、评分维度数、创建时间
   - 新建按钮 → Modal 表单：
     - 名称、System Prompt（TextArea）、
     - 评分维度（动态添加/删除：名称+描述+权重）、
     - LLM 模型选择
   - 编辑按钮 → 同上 Modal
   - 删除按钮 → 确认弹窗

2. AI 预审监控页 (pages/AiReview/index.tsx)：
   - 任务队列 (TaskQueue)：展示待审核的任务
   - 审核列表：task名称、题目、Agent、verdict标签（颜色区分）、评分、状态、耗时
   - 筛选：按 Agent、按 verdict、按 status
   - 每条点击展开详情：
     - 提交内容 (SubmittedContent)：标注员的原始提交
     - AI 评分 (DimensionScores)：多维度评分可视化
     - AI 评语 (AiComments)：summary + 各维度评语
     - Prompt 展示：使用的 prompt_template + prompt_vars
     - 处理日志 (ProcessLogs)：时间线
   - 失败重跑按钮

请按后端模型→API→前端页面的顺序实现。
```

## AI 的回答

```
AiAgent model (system_prompt/scoring_dimensions JSON/llm_model), agent CRUD + status API, AiReview model (verdict/dimensions JSON/overall_score/prompt_template/prompt_vars/status), ai-reviews list/detail/rerun/stats APIs, AiConfigs page (agent table + create/edit modal), AiReview page (queue + detail with dimension scores + AI comments + prompt display + process logs)
```
