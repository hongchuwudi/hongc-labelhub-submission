## 6. 部署方案

```yaml
# docker-compose.yml 服务清单
services:
  db:        # MySQL 8.0              — 端口 3306
  redis:     # Redis 7-alpine         — 端口 6379
  rabbitmq:  # RabbitMQ 3-management  — 端口 5672/15672
  backend:   # FastAPI + AgentPool    — 端口 8000
  frontend:  # Vite dev / Nginx       — 端口 5173
```

单机部署，一个 `docker-compose up -d` 启动全部服务。
