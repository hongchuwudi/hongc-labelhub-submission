# LabelHub 技术栈总览

## 整体架构

```
React 19 SPA (Vite) → Nginx (SSL) → FastAPI (Gunicorn) → MySQL/Redis/RabbitMQ → LLM API
```

## 前端技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| React | 19 | UI 框架，函数组件 + Hooks |
| TypeScript | 6 (strict) | 类型安全，零 `any` |
| Vite | 8 | 构建工具，HMR 热更新 |
| Ant Design | 6 | UI 组件库 |
| Formily | 2.x | JSON Schema 驱动动态表单渲染 |
| @dnd-kit | latest | 拖拽排序（Schema 设计器 + Tab 栏） |
| Zustand | 5 | 全局状态管理（user / tokens / authReady） |
| React Router | 7 | SPA 路由 |
| ECharts | 6 | 数据看板图表 |
| Axios | latest | HTTP 客户端，拦截器自动 JWT 续签 |
| Day.js | latest | 日期处理，UTC→本地时区转换 |

## 基础设施

| 组件 | 版本 | 用途 |
|------|------|------|
| MySQL | 8.0 (Docker) | 主数据库，12 张表 |
| Redis | 7 (Docker) | 分布式锁（抢单 SETNX）+ FIFO 队列 + Token 白名单 |
| RabbitMQ | 3 (Docker) | AI 审核队列 + 导出队列 |
| Nginx | Alpine (Docker) | SSL 终端 + 反向代理 + 静态文件服务 |
| Let's Encrypt | — | 免费 SSL 证书 |

## 部署架构

```
systemd(labelhub.service) → Gunicorn(2 workers) → FastAPI App
Nginx Docker(nginx-https) → /api/ → proxy_pass 172.17.0.1:8000
                            → /     → 静态文件 SPA
```

- 后端：宿主机直接运行，systemd 管理进程，SCP 上传代码部署
- 前端：Vite build → 静态文件 → Nginx 托管
- 中间件：全部 Docker 容器运行，数据卷持久化
- SSL：Let's Encrypt + certbot 自动续期
